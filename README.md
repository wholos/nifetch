# nifetch
# This Fetch can be used on Linux distributions, as well as on FreeBSD, OpenBSD, NetBSD
[![Nim](https://img.shields.io/badge/Nim-%23FFE953.svg?&logo=nim&logoColor=white)](#)

How to build?
``` bash
make all
```

How to install?
``` bash
make install
```

How to uninstall?
``` bash
make uninstall
```
